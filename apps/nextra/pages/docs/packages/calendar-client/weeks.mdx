# Weeks

This component alone is useful for paging through weeks of a month (common calendar as you know it).

## Translations

List of used translations, defaults to English. All translations comes from **calendar** namespace.

| Label                        | Description                        |
|:-----------------------------|------------------------------------|
| weeks.previous-month.tooltip | Tooltip over previous month button |
| weeks.next-month.tooltip     | Tooltip over next month button     |
| weeks.previous-event.tooltip | Tooltip over previous event button |
| weeks.next-event.tooltip     | Tooltip over next event button     |

## Minimal example

<SourceExample
    example={<Shell>
        <DefaultWeeks/>
    </Shell>}
>
```tsx
import {
    Weeks,
    WeeksProvider
}                from "@leight/calendar-client";
import {type FC} from "react";

export interface IDefaultWeeksProps {
}

export const DefaultWeeks: FC<IDefaultWeeksProps> = () => {
    /**
     * Here you can use WeeksProvider or CalendarProvider.
     *
     * CalendarProvider provides all stuff needed for the whole Calendar (weeks/months/years/...).
     */
    return <WeeksProvider>
        <Weeks/>
    </WeeksProvider>;
};
```
</SourceExample>
