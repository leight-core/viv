# @leight/table-client

Here you can find a way how to work with UI tables built on top of [Mantine Table](https://mantine.dev/core/table/) and [`Source`](../concepts/source).

## Installation

<Tabs items={['npm', 'pnpm', 'yarn']}>
    <Tab>
        ```bash copy filename="npm"
        npm i @leight/table-client
        ```
    </Tab>
    <Tab>
        ```bash copy filename="pnpm"
        pnpm i @leight/table-client
        ```
    </Tab>
    <Tab>
        ```bash copy filename="yarn"
        yarn add @leight/table-client
        ```
    </Tab>
</Tabs>

## Usage

Wraps [Mantine Table](https://mantine.dev/core/table/) with quite simpler usage, connects to [`Source`](../concepts/source) and also supports [i18next](https://www.i18next.com/) by default.
