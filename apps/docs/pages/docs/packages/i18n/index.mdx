# @leight/i18n

This library provides common localization stuff for the client and server. It also contains some interesting methods for generating calendar
data.

## Installation

<Tabs items={['npm', 'pnpm', 'yarn']}>
    <Tab>
        ```bash copy filename="npm"
        npm i @leight/i18n
        ```
    </Tab>
    <Tab>
        ```bash copy filename="pnpm"
        pnpm i @leight/i18n
        ```
    </Tab>
    <Tab>
        ```bash copy filename="yarn"
        yarn add @leight/i18n
        ```
    </Tab>
</Tabs>
