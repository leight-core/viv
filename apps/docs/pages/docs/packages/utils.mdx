## API

Each section here is exported from the package.

### chain()

Useful when you need to rotate through values, for example `true{:js}` -> `false{:js}` -> `undefined{:js}`.

Initial use case for this method was rotating sort order in tables (`"asc"{:js}` -> `"desc":{:js}` -> `undefined{:js}`).

```ts
import {chain} from "@leight/utils";

/**
 * Next value: `false`
 */
console.log("Next value", chain(true, [true, false, undefined]));
/**
 * Next value: `A`
 */
console.log("Next value", chain("C", ["A", "B", "C"]));
```

### cleanOf()

<Callout type="info" emoji="ℹ️">
    This method is proxy to [fast-clean](https://www.npmjs.com/package/fast-clean) package.
</Callout>

Takes an object as an input and removed all `undefined{:js}` properties while preserving `null{:js}`s.

Second parameter of the method is [options object](https://www.npmjs.com/package/fast-clean#options) from `fast-clean`.

```ts
import {cleanOf} from "@leight/utils";

/**
 * Only `wipeMe` will be removed.
 */
console.log("cleanOf", cleanOf({
    foo: "true",
    bar: false,
    hello: null,
    wipeMe: undefined,
}));
```
